Welcome to My Portfolio

Thank you for visiting! This webpage has been built using Angular for the frontend, and it utilizes server-side rendering (SSR) for improved performance and SEO.

The following tools and resources have been used to build this webpage:

  • Angular - A platform for building mobile and desktop web applications.
  • Angular Universal - Provides server-side rendering for Angular applications.
  • Express - A minimal and flexible Node.js web application framework.
  • Tailwind CSS - A utility-first CSS framework for building custom designs.
  • RxJS - A library for reactive programming using Observables.
  • unDraw - A collection of open-source illustrations for various design projects.
  • Pexels - A free stock photo and video website for finding high-quality images and videos.
